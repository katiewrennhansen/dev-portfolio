Healthy Plate

This project was built to help individuals interested in expanding the diversity of their diets explore new food options. It allows the user to search for any food and learn more about it's origins and nutritional information. It provides the user with recipes including the searched food and allows for filtration based on dietary needs.

Technologies Used:

  • HTML
  • CSS
  • jQuery
  • RESTful APIs
