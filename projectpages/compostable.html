Compostable

This crowdsourced application aims to connect urban people hoping to compost with composts in their area. Users can list their own compost, search for other composts on an interactive map and message other users that have listed composts.

Technologies Used:

  • React
  • CSS
  • Node.js
  • SQL
  • PostgreSQL
