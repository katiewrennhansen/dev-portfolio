Healthy Plate

This project was built to help individuals interested in expanding the diversity of their diets explore new food options. It allows the user to search for any food and learn more about it's origins and nutritional information. It also provides the user with recipes that incorperate the searched food and allows for recipe filtration based on various dietary needs.

Technologies Used:

  • HTML
  • CSS
  • jQuery
  • APIs
